This plugin establishes a common communication for customers and suppliers where suppliers can share prices and details of a single product.

== Description ==

Features:-
1) Product quote request submission for logged-in users only.
2) Provides a customizable form for users to submit a Product Quote.
3) Display product add-to-quote button in product loop and shop page.
4) There are three option to set product quote button (1) add-to-quote with add-to-cart button (2) add-to-quote button without add-to-cart button and (3) default.
5) Display user all request quote detail in my-account area which is submitted with status.
6) Admin acn delete request of product quote and change status of Quote.
7) User get gmail whene Request status is update by admin.
8) Admin can Add To Quote Button for all page from Woocommerce->Setting->Products->Woocommerce Quote.

== Screenshots ==
1. Admin can configure add to quote button for particular product in product tab ‘ASPL Quotation’.

3. Admin can see all detail in edit page of request quote like customer profile ,customer contact number ,customer email id ,customer request message and comparison between original and expected product price.
